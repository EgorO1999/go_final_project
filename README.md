Это веб-приложение для управления задачами. Оно позволяет:
- добавить задачу;
- получить список задач;
- удалить задачу;
- получить параметры задачи;
- изменить параметры задачи;
- отметить задачу как выполненную.

Список выполненных заданий:
- реализованна возможность определять путь к файлу базы данных через переменную окружения. Для этого сервер должен получать значение переменной окружения TODO_DBFILE
- реализованна возможность определять извне порт при запуске сервера. Если существует переменная окружения TODO_PORT
- считать повторения по дням недели и определенному числу месяца
- добавленав возможность искать определенную задачу через search

Инструкция по локальному запуску:

Создайте .env файл или установите переменные напрямую:

TODO_PASSWORD = "пароль"
TODO_DBFILE = "путь_к_бд"
TODO_PORT = "порт"

Запустите сервер:

go run ./todoapp

Откройте в браузере:

http://localhost:порт(по умолчанию 7540)
